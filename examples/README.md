# Examples

## Before/After Comparisons

Three scenarios showing the same prompt answered by generic AI vs. a Digital Twin. Each file includes the prompt, both outputs, and a line-by-line breakdown of what changed.

All three use the same Founder/CEO profile so you can see the voice hold across contexts (email, Slack, pitch).

- [Follow-up Email](before-after/follow-up-email.md) — client ghosted after proposal
- [Slack Pushback](before-after/slack-pushback.md) — disagreeing with a collaborator's approach
- [Project Proposal](before-after/project-proposal.md) — pitching a new engagement

## Sample Profiles

Complete Digital Twin profiles for three roles. These use real extraction patterns (anonymized), not templates. Each includes Communication DNA, Tone Guidelines, Decision Logic, Signature Phrases, Anti-Patterns, Knowledge Map, and the surprising pattern the extraction surfaced.

- [Founder/CEO](sample-profiles/founder-ceo.md) — 60 files, 27K words, Layer 3
- [Creative Director](sample-profiles/creative-director.md) — 45 files, 18K words, Layer 3
- [Technical Lead](sample-profiles/technical-lead.md) — 30 files, 22K words, Layer 2
